# 创建集群<a name="ZH-CN_TOPIC_0000001098816618"></a>

## 功能介绍<a name="scd66d518a80c4ea88a94f13aa57bb1e1"></a>

该接口用于创建集群。

集群必须要运行在VPC之内，创建集群前，您需要先创建VPC，并获取VPC和子网的id。

该接口为异步接口，创建集群需要10～15分钟。

## 调试<a name="section16191247193116"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DWS&api=CreateCluster)调试该接口。

## URI<a name="see3b10011fa94758b8bc201af3e10d11"></a>

-   URI 格式

    ```
    POST /v1.0/{project_id}/clusters
    ```
