# 创建集群<a name="dws_02_0020"></a>

## 功能介绍<a name="scd66d518a80c4ea88a94f13aa57bb1e1"></a>

该接口用于创建集群。

集群必须要运行在VPC之内，创建集群前，您需要先创建VPC，并获取VPC和子网的ID。

该接口为异步接口，创建集群需要10～15分钟。

## URI<a name="see3b10011fa94758b8bc201af3e10d11"></a>

-   URI 格式
