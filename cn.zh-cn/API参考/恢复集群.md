# 恢复集群<a name="ZH-CN_TOPIC_0000001145816475"></a>

## 功能介绍<a name="s4bb187ea1af94154b7c794fc9f01b061"></a>

该接口用于使用快照恢复集群。

## 调试<a name="section16191247193116"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DWS&api=RestoreCluster)调试该接口。

-   URI 格式

    ```
    POST /v1.0/{project_id}/snapshots/{snapshot_id}/actions
    ```
