# 异常响应<a name="dws_02_0029"></a>

DWS API提供了统一的异常响应，它包含错误代码和指定消息的异常细节。

## 响应<a name="s52aa22886a1b455db4b11febafcc4d02"></a>

-   响应样例

    ```
    STATUS CODE 500
    {
        "errCode": "DWS.0001",
        "externalMessage": " 参数错误！"
    }
    ```
